# 🌟 Pattern Generator and Number Analyzer

## 📖 Project Overview

The **Pattern Generator and Number Analyzer** is a Python console application designed to practice basic programming concepts. It allows users to generate a star pattern, analyze numbers within a given range, and understand the use of Python control statements through a simple menu-driven interface.

---

## 🚀 Features

- Generate a right-angled triangle using `*`
- Check whether numbers are even or odd
- Calculate the sum of numbers in a specified range
- Interactive menu for repeated operations
- Demonstrates the use of:
  - `for` loops
  - `while` loops
  - `if-elif-else`
  - `break`
  - `continue`
  - `pass`

---

## 🛠️ Requirements

- Python 3.x

## ▶️ How to Run

1. Download or clone this project.
2. Open a terminal in the project folder.
3. Run the program using:

